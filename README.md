## Files

## How to Deploy (FREE)

1.  Create a GitHub account
2.  Create a new repository
3.  Upload the files
4.  Go to **Settings → Pages**
5.  Enable GitHub Pages

Your website will be live like:

https://yourusername.github.io/repository-name
